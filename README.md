Projektziel:

Named-entity recognition von deutschen Rechtsnormen - in der einfachen Form von z.B. Art. 33 GG oder § 15a StVO - mit Hilfe von regulärer Ausdrücke (RegEx) und die Rückgabe ihrer Anzahl.
